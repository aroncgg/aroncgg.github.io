CONCEPTO DE VIRTUALIZACIÓN

Describe la capacidad de algunas computadoras de utilizar un programa o un conjunto de programas para imitar las características físicas de otra computadora o de un conjunto de computadoras, lo que da lugar a un sistema informático virtual.

TIPOS DE VIRTUALIZACIÓN

DOCKER

Es un sistema operativo para contenedores. De manera similar a cómo una máquina virtual virtualiza (elimina la necesidad de administrar directamente) el hardware del servidor, los contenedores virtualizan el sistema operativo de un servidor.
